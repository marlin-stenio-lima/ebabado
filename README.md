# Acess Vibe

Sistema de gestão para eventos, PDV e Cardápio Digital.

## Estrutura
- `web/`: Aplicação Frontend (Next.js + TailwindCSS + Shadcn/UI)

## Como Iniciar

1. Navegue para a pasta `web`:
   ```bash
   cd web
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Stack
- Next.js (App Router)
- React
- TailwindCSS
- Shadcn/UI
- Supabase
